# Grounded Cafe & Pub Website

Official website for **Grounded Cafe & Pub**.

## Project Status

🚧 Currently in development.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4

## Development

Start the development server:

```bash
npm run dev
```

Open:

http://localhost:3000

- Simple, Scalable, Intentional
